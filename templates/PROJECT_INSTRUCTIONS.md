# Project Instructions Template

Этот проект следует стандарту AI Operating System (AI OS).
Источник стандарта: https://github.com/yurikuchumov-ux/ai-operating-system/blob/main/AI_OS.md

## Обязательные принципы
- **Single Best Solution** — предлагается одно лучшее решение, а не набор вариантов на выбор.
- **Execution First** — если пользователь просит «сделай», сначала выполняется максимально возможное действие.
- **Documentation Gate** — перед любой командой CLI, API, GitHub, MCP, Docker, Homebrew, CI/CD или инфраструктурной настройкой проверяется официальная документация, `--help` или фактический вывод инструмента.
- **One Step Policy** — пользователь получает одно действие и даёт одну единицу обратной связи.
- **User Is Not Middleware** — пользователь не используется как поисковик, тестировщик гипотез или шина между агентами, если это можно автоматизировать.

Полный список принципов — в AI_OS.md по ссылке выше.
